# 设定档案馆 / World Archive

一个本地优先的世界观与 OC 设定管理工具，用来整理角色、地点、组织、物品、事件、种族、世界观与笔记，以及它们之间的关联。

## 在线体验

[https://www.lorefolio.cn/](https://www.lorefolio.cn/)

（备用地址：[https://lorefolio.vercel.app/](https://lorefolio.vercel.app/)）

## 适合用来做什么

- 小说与故事的世界观设定整理
- OC 角色档案管理
- 地点、势力、道具的系统化记录
- 种族 / 生物设定整理
- 事件 / 历史纪年整理
- 世界观规则、魔法体系、历史与文化归档
- 创作笔记与灵感备忘
- 本地个人创作工作台

## 当前功能

### 世界项目

- 创建多个独立的世界 / 项目
- 每个世界内管理专属的设定条目

### 角色档案

- 外貌、性格、代称、身份、状态、代表台词
- 能力与技能、目标与执念
- 背景故事与补充设定
- 所属组织、常驻地点、种族的条目引用
- 角色间关系图谱
- 角色关联：参与事件、代表种族、持有物品

### 地点档案

- 地点分类、当前状态
- 环境氛围、标志性地标与建筑
- 历史沿革、出入方式与限制
- 上级地点层级、主要管辖组织
- 地图坐标（支持在世界地图中查看和拖拽定位）

### 组织 / 势力档案

- 组织类型、当前状态
- 理念与目标、内部结构、影响力与势力范围
- 历史沿革、创作备注
- 上级组织、总部地点
- 自动派生显示成员角色、管辖地点、下属组织

### 物品 / 道具档案

- 物品类型、当前状态
- 外观描述、功能用途、材质构造
- 来源出处、历史沿革
- 使用限制与副作用、创作备注
- 当前持有者、当前所在地、制造 / 所属组织的条目引用

### 事件档案

- 事件类型、进行状态
- 自由文本纪年与时间范围
- 发生地点、主要相关组织
- 参与角色、涉及组织、相关物品（均支持多选引用）
- 起因、经过、结果三段叙事结构
- 影响、后续发展、创作备注
- 时间线排序值

### 种族档案

- 种族分类、状态
- 主要栖息地（地点引用）
- 相关组织、代表角色（均支持多选引用）
- 外貌特征、生理 / 生态特征、能力与天赋
- 文化 / 习俗 / 社会结构、历史与起源
- 分布范围、与其他族群关系、创作备注

### 世界观档案

- 世界观分类、状态（草稿 / 稳定 / 废弃）
- 核心概念、世界规则、宇宙观 / 起源
- 历史背景、魔法 / 科技体系、文化制度、核心冲突
- 关联地点、组织、种族、事件、物品（均支持多选引用）
- 创作备注

### 笔记

- 通用富文本条目，适合创作备忘、剧情草稿和随手记录
- 支持标签、相关条目、收藏与置顶

### 工作台与视图

- **全局搜索**：跨分类搜索当前世界的标题、摘要与标签，支持按条目类型筛选，搜索结果支持高亮和命中来源提示
- **世界概览**：条目数量统计、档案完整度、最近更新、收藏/置顶、热门标签、事件与纪年摘要
- **时间线**：基于事件档案的垂直时间线，按纪年分组，支持搜索和标签筛选
- **世界地图**：支持上传自定义地图底图，在地图上标注地点、拖拽地点调整位置、点击空白区域创建地点，支持缩放与平移。未设置坐标的地点自动临时分散展示，地点列表支持快速定位与居中
- **角色关系图谱**：以可视化图谱展示角色之间的关系网络
- 电脑端 Sidebar 提供工作台导航入口
- 手机端 TopBar 提供项目工具菜单（全局搜索、地图、时间线、关系图谱、概览）

### 通用能力

- 富文本正文编辑（支持正文插图）
- 封面图与图集
- 标签系统
- 相关条目关联
- 收藏与置顶
- JSON 数据导出（含导出前数据统计确认）
- JSON 备份文件校验
- 备份文件导入恢复（含导入前条目预览）
- 导入前自动下载当前数据备份
- 删除世界项目前自动下载完整数据备份
- 首次访问提供示例世界（可自由修改或删除）
- 最近备份时间提醒
- 返回顶部按钮（桌面端与移动端）
- PWA 可安装（支持添加到主屏幕）

## 数据存储说明

本项目为本地优先应用，**设定数据保存在当前浏览器的 localStorage 中**，不会上传或同步到任何服务器。

以下操作可能导致看不到原来的数据：

- 清除浏览器数据（缓存 / Cookie / 站点数据）
- 更换浏览器或更换设备
- 使用无痕 / 隐私模式

**重要建议：**

- 定期使用页面上的「导出数据」功能保存 JSON 备份到本地文件。
- 导入备份会覆盖当前浏览器中的数据，导入前系统会自动下载一份导入前数据作为备份。
- 经常导出备份是保护创作资料最可靠的方式。

## 项目状态

当前已完成基础可用版本，支持角色、地点、组织、物品、事件、种族、世界观、笔记八类档案与本地备份恢复流程，并具备全局搜索、世界概览、时间线、世界地图和角色关系图谱等视图。项目仍在持续优化移动端体验、数据安全和创作工作流。
